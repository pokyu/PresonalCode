## Python-Markdown Extra

这是对于模仿 PHP Markdown Extra 的 python-markdown 扩展的一个合辑
支持下面这些扩展：
```
Abbreviations           缩写
Attribute Lists         属性列表
Definition Lists        定义列表
Fenced Code Blocks      代码块
Footnotes               脚注
Tables                  表格
Smart Strong            加粗
```
